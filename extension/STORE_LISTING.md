## Name

Willing to Contribute - Open Source Issue Finder

## Short Description (132자 이내)

Discover beginner-friendly GitHub issues. Track repos, get notifications, and find your first open source contribution — right from your browser toolbar.

## Detailed Description (English)

**Willing to Contribute** helps developers of all levels discover and track beginner-friendly GitHub issues without leaving their browser.

### Key Features

- **Issue Discovery** — Browse open `good first issue`, `help wanted`, and custom-labeled issues from multiple repositories in one place.
- **Repository Tracking** — Add any public GitHub repository by URL or owner/name. Your list is saved and synced automatically.
- **Browser Notifications** — Get notified when new issues matching your preferences appear. Configure notification frequency to your liking.
- **GitHub Page Integration** — A "Track Repository" button is injected directly into GitHub repository pages so you can add repos without opening the popup.
- **Cross-Device Sync** — Settings sync across Chrome instances via `chrome.storage.sync` (requires signing in to Chrome with your Google account).
- **GitHub Token Support** — Optionally provide a personal GitHub API token to increase rate limits and access private repositories.
- **Multilingual** — Full English and Korean (한국어) interface support.

### How to Use

1. Install the extension and click the toolbar icon.
2. Go to the **Repositories** tab and add GitHub repositories you want to follow.
3. Switch to the **Issues** tab to view open beginner-friendly issues.
4. Configure label filters and notification frequency in the **Settings** tab.
5. Browse GitHub normally — a "Track" button will appear on repository pages automatically.

### Privacy

All data is stored locally on your device using `chrome.storage.local` and `chrome.storage.sync`. No data is sent to external servers. GitHub API calls are made directly from your browser using your own token.

Full privacy policy: <https://willing-to-contribute.vercel.app/privacy>

## 한국어 설명

**Willing to Contribute**는 개발자들이 입문자 친화적인 GitHub 이슈를 브라우저에서 바로 탐색하고 추적할 수 있도록 도와주는 확장 프로그램입니다.

### 주요 기능

- **이슈 탐색** — `good first issue`, `help wanted` 등 입문자용 라벨이 붙은 이슈를 여러 레포지토리에서 한 곳에 모아 확인.
- **레포지토리 추적** — URL 또는 소유자/이름 형식으로 공개 GitHub 레포지토리를 추가하고 목록을 저장.
- **브라우저 알림** — 새 이슈가 등록되면 알림을 받고, 알림 빈도를 자유롭게 설정.
- **GitHub 페이지 통합** — GitHub 레포지토리 페이지에 "Track" 버튼이 자동으로 추가되어 팝업 없이 즉시 추가 가능.
- **기기 간 동기화** — `chrome.storage.sync`를 통해 Chrome에 로그인된 모든 기기에 설정이 자동 동기화.
- **GitHub 토큰 지원** — 개인 API 토큰을 등록하면 API 요청 한도가 늘어나고 비공개 레포지토리에도 접근 가능.
- **다국어 지원** — 영어/한국어 인터페이스 완벽 지원.

### 사용 방법

1. 확장 프로그램을 설치하고 툴바 아이콘을 클릭하세요.
2. **Repositories** 탭에서 팔로우할 GitHub 레포지토리를 추가하세요.
3. **Issues** 탭에서 열린 입문자 친화적 이슈를 확인하세요.
4. **Settings** 탭에서 라벨 필터 및 알림 빈도를 설정하세요.
5. GitHub를 평소처럼 탐색하면 레포지토리 페이지에 "Track" 버튼이 자동으로 나타납니다.

### 개인정보 보호

모든 데이터는 `chrome.storage.local` 및 `chrome.storage.sync`를 통해 기기에 로컬로 저장됩니다. 외부 서버로 데이터를 전송하지 않으며, GitHub API는 사용자의 토큰으로 브라우저에서 직접 호출합니다.

전체 개인정보처리방침: <https://willing-to-contribute.vercel.app/privacy>
